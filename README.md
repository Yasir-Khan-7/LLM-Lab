# LLM-Lab

LLM-Lab is an experimental repository for building and testing Large Language Model (LLM) powered tools, NLP workflows, and website scraping plus brochure generation workflows.

## What is included

- Web content extraction for scraping text and links from websites.
- Link relevance selection to find company pages useful for marketing or investor materials.
- An LLM-based brochure generation workflow that creates concise summaries from site content.
- Notebook experiments exploring NLP concepts and tokenization.

## Key features

- website scraping and HTML extraction
- LLM-guided link selection for relevance
- brochure-style content generation from website data


## Getting started

1. Clone the repository.
2. Create and activate a Python virtual environment.
3. Install the required Python libraries.
4. Add your API key to an environment configuration file.
5. Run the main brochure generation workflow.

## Notes

- This repo is designed for experimentation and learning, not production deployment.
- It combines web scraping, link filtering, and LLM summarization.
- Adjust the prompts and URL targets to explore different websites.
